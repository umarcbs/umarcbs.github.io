OUR MOTIVES

We aim to make flora flourish in the Valley.
"An initiative to help secure the Flora of Valley."

Plants Care

We aim to make the details of medicinal plants easily available so that people can protect them.

Medicinal Uses

Our aim is to help mankind in getting cured from diseases; we are disclosing all the ethno-medicinal uses of plants.

Helping Researchers & Botanists

We also aim to help them know what is already known so that they focus on revealing new compounds and treatments.

This website is used for uploading a database that we named as "MAPs" of J&K - (Medicinal & Aromatic Plants of Jammu & Kashmir).

Which programming languages are used in this website?

This website uses HTML5, CSS3, JavaScript, PHP programming languages. We also used BootStrap and JQuery that are scripts made from CSS and JavaScript, respectively. We also used Datatables in one section of this website for entering the data about plants in a tabular form wherein we only deal with textual data.

What is this database about?

This database is all about the Medicinal & Aromatic Plants of Jammu & Kashmir. It contains details about the plants of J&K like name of plant, family, etc.

What attributes are used in this database?

The attributes/features that we used in this database are:

  • Botanical Name Author
  • Vernicular Name
  • Family
  • Habit
  • Part(s) used
  • Status
  • District, etc.
